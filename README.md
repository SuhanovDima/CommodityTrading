# Invoice Generator

Java-приложение для генерации счетов (invoice) по заданным шаблонам в формате Excel.

## Описание

Программа предназначена для автоматической генерации счетов на основе предопределённых шаблонов Excel. 
Позволяет заполнять шаблоны данными и сохранять результат в виде готовых файлов invoice.

## Требования

- Java 21 или выше
- Maven для сборки проекта

## Структура проекта

```
├── src/main/java/commodity/trading/   # Исходный код приложения
├── pom.xml                            # Конфигурация Maven
└── README.md                          # Документация проекта
```

## Сборка и запуск

### Сборка проекта

```bash
mvn clean install
```

### Запуск приложения

```bash
java -jar target/CommodityTrading-1.0-SNAPSHOT.jar
```

## Использование

Приложение генерирует invoice файлы в формате Excel на основе шаблонов. 
Для работы необходимо:

1. Подготовить шаблоны Excel с необходимыми полями
2. Передать данные для заполнения шаблона
3. Получить готовый invoice файл

## Технологии

- **Java 21** - основной язык разработки
- **Apache POI** - библиотека для работы с Excel файлами (рекомендуется добавить в зависимости)
- **Maven** - система сборки проекта

## Добавление зависимостей

Для работы с Excel файлами рекомендуется добавить Apache POI в `pom.xml`:

```xml
<dependencies>
    <dependency>
        <groupId>org.apache.poi</groupId>
        <artifactId>poi-ooxml</artifactId>
        <version>5.2.5</version>
    </dependency>
</dependencies>
```
